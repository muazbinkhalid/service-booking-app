# 🚀 Service Booking App (Flutter + Firebase)

A modern **service marketplace mobile app** built using Flutter and Firebase, where users can book services and providers can manage their business.

---

## 📱 Features

### 👤 User Side
- Browse services (Cleaning, Plumbing, Electrician)
- View service providers
- Book services with date selection
- Real-time booking system
- Clean and modern UI

### 🧑‍🔧 Provider Side
- Provider dashboard
- Add & manage services
- View earnings
- Ratings system
- Profile management

---

## 🔥 Tech Stack

- **Flutter** (Frontend)
- **Firebase Authentication**
- **Cloud Firestore**
- **Material UI + Custom Design**
